## Essential Machine Learning Concepts

Here are the most important terms you need to understand when working with machine learning:

### Core Concepts

- **Model** — The trained software that makes predictions or generates content.
- **Features** — The input variables used by the model to make predictions (also called independent variables).
- **Target / Label** — The output we want the model to predict (also called dependent variable).
- **Training** — The process of teaching the model by showing it data and adjusting its internal parameters.
- **Inference** — Using a trained model to make predictions on new, unseen data.

### Key Challenges

- **Overfitting** — The model performs very well on training data but poorly on new data (it memorizes instead of learning).
- **Underfitting** — The model is too simple and fails to capture important patterns in the data.
- **Bias-Variance Tradeoff** — Balancing underfitting (high bias) and overfitting (high variance).

### Data Handling

- **Train Set** — Data used to train the model.
- **Validation Set** — Data used to tune hyperparameters and evaluate during development.
- **Test Set** — Unseen data used for final evaluation of model performance.

### Other Important Terms

- **Loss Function** — Measures how far the model's predictions are from the actual values.
- **Optimizer** — Algorithm that minimizes the loss function (e.g., Gradient Descent, Adam).
- **Hyperparameters** — Settings you configure before training (learning rate, number of trees, etc.).
- **Feature Engineering** — Creating new useful features from existing data to improve model performance.

## Common Evaluation Metrics

| Task Type       | Common Metrics                          |
|-----------------|-----------------------------------------|
| **Classification** | Accuracy, Precision, Recall, F1-Score, ROC-AUC |
| **Regression**     | MSE, RMSE, MAE, R² Score               |
| **Clustering**     | Silhouette Score, Inertia              |
