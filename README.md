# ML-Fundamentals-p1

A practical, straight-forward machine learning repository focused on building real understanding — not just running tutorials.

This project is designed to help you go from:
> “I can run ML code” → “I understand what’s actually happening and can improve it”

---

## 🎯 Purpose

Most ML resources fall into two extremes:
- Too theoretical (hard to apply)
- Too abstracted (copy-paste without understanding)

This repo sits in the middle:
- ✅ Practical implementations
- ✅ Clear explanations
- ✅ Real-world thinking (including betting models / predictive systems)
- ✅ Focus on fundamentals that actually matter

---

## 🧠 What You’ll Learn

### Core Concepts
- What machine learning actually is (beyond buzzwords)
- Difference between:
  - Features vs labels
  - Parameters vs hyperparameters
- Overfitting vs underfitting
- Model evaluation (accuracy vs real-world usefulness)
