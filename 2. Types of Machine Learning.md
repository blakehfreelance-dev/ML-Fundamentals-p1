# 2. Types of Machine Learning

**Part 2 of ML Fundamentals Series**

---

## Main Types of Machine Learning

### 1. Supervised Learning
The model learns from **labeled data** (inputs + correct outputs).

**Goal:** Learn to predict the output for new, unseen data.

**Common Tasks:**
- **Classification** → Predicting categories (e.g., spam/not spam, cat/dog)
- **Regression** → Predicting continuous values (e.g., house price, temperature)

**Popular Algorithms:**
- Linear Regression
- Logistic Regression
- Decision Trees & Random Forest
- Support Vector Machines (SVM)
- Neural Networks

---

### 2. Unsupervised Learning
The model learns from **unlabeled data**.

**Goal:** Discover hidden patterns, structures, or groupings.

**Common Tasks:**
- Clustering (group similar data)
- Dimensionality Reduction
- Anomaly Detection

**Popular Algorithms:**
- K-Means Clustering
- Hierarchical Clustering
- PCA (Principal Component Analysis)
- Autoencoders

---

### 3. Reinforcement Learning (RL)
An agent learns by interacting with an environment through **trial and error** and rewards.

**Use Cases:** Game AI (AlphaGo), robotics, autonomous vehicles, recommendation systems.

---

### 4. Other Types
- **Semi-Supervised Learning** — Small labeled + large unlabeled data
- **Self-Supervised Learning** — Model creates its own labels (very popular in LLMs & modern vision)
