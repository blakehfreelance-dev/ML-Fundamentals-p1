## What is Supervised Learning?

Supervised Learning is the most common type of machine learning.  
The model is trained on **labeled data** — meaning each training example has both input features and the correct output (label).

**Goal:** Learn the mapping from inputs to outputs so it can accurately predict the output for new, unseen data.

---

## Main Types of Supervised Learning Tasks

### 1. Regression
Predicts **continuous numerical values**.

**Examples:**
- House price prediction
- Stock price forecasting
- Temperature prediction

**Popular Algorithms:**
- Linear Regression
- Polynomial Regression
- Random Forest Regressor
- Gradient Boosting (XGBoost, LightGBM, CatBoost)
- Neural Networks

### 2. Classification
Predicts **discrete categories** or classes.

**Examples:**
- Email spam detection (spam / not spam)
- Image classification (cat / dog / bird)
- Disease diagnosis (positive / negative)

**Popular Algorithms:**
- Logistic Regression
- Decision Trees
- Random Forest Classifier
- Support Vector Machines (SVM)
- k-Nearest Neighbors (k-NN)
- Gradient Boosting Classifiers
- Neural Networks / Deep Learning

---

## Simple Comparison of Popular Algorithms

| Algorithm              | Best For                  | Advantages                          | Disadvantages                     | Type          |
|------------------------|---------------------------|-------------------------------------|-----------------------------------|---------------|
| Linear Regression      | Simple relationships      | Very fast, easy to interpret        | Cannot handle complex patterns    | Regression    |
| Logistic Regression    | Binary classification     | Simple & fast                       | Limited to linear boundaries      | Classification|
| Decision Trees         | Mixed data types          | Easy to understand                  | Prone to overfitting              | Both          |
| Random Forest          | Most tabular data         | Accurate, less overfitting          | Slower & harder to interpret      | Both          |
| Gradient Boosting      | High performance          | Often best accuracy on tabular data | Requires more tuning              | Both          |
| Neural Networks        | Images, text, complex data| Very powerful                       | Needs lots of data & computation  | Both          |

---

## Basic Workflow for Supervised Learning

1. Collect and clean labeled data
2. Split data (Train / Validation / Test)
3. Choose and train a model
4. Evaluate using appropriate metrics
5. Tune hyperparameters
6. Deploy the best model
